- H2 database (username/password - sa/password)
- Hibernate

## Installation
On console execute the command:
```
mvn install
```
```
java -jar ApplaudoTest-0.0.1-SNAPSHOT.jar
```
And execute the requests from file ApplaudoTest.postman_collection.json through Postman.
